## 1. La phrase et ses constituants

### Formule fondamentale

**P = [GS + GV] (+ GC)**

- **P** = phrase
- **GS** = groupe sujet (obligatoire)
- **GV** = groupe verbal (obligatoire)
- **GC** = groupe circonstanciel (facultatif)

Les crochets indiquent la solidarité entre GS et GV ; les parenthèses indiquent le caractère facultatif du GC.

### Exemples

| Phrase | GS | GV | GC |
|---|---|---|---|
| *Le facteur distribue le courrier.* | le facteur | distribue le courrier | — |
| *Le facteur distribue le courrier à huit heures.* | le facteur | distribue le courrier | à huit heures |

### Principe fondamental

Seules **deux fonctions** existent au niveau de la phrase :
- la **fonction sujet** (relation entre GS et GV)
- la **fonction complément circonstanciel** (relation entre [GS + GV] et GC)

Toutes les autres fonctions (COD, épithète, etc.) n'existent qu'**à l'intérieur** des groupes qui forment la phrase.

## 2. Les fonctions grammaticales

### 2.1 La fonction sujet

Le **groupe sujet (GS)** règle l'accord du verbe. Il peut être constitué de :

| Nature du GS | Exemple |
|---|---|
| Groupe nominal (GN) | *Le facteur* distribue le courrier. |
| Pronom personnel | *Il* distribue le courrier. |
| Infinitif | *Jouer* me repose. |
| Proposition subordonnée complétive | *Qu'elle soit en retard* me surprend. |
| Proposition subordonnée relative substantive | *Qui se ressemble* s'assemble. |
| Proposition subordonnée interrogative partielle | *Pourquoi elle a voulu partir* demeure inexpliqué. |

### 2.2 Le groupe verbal et ses fonctions internes

Le **groupe verbal (GV)** se compose d'un verbe et de constituants dont la fonction est interne au GV.

#### Les principaux types de GV

| Structure | Exemple |
|---|---|
| GV = V | *Le facteur travaille.* |
| GV = V + GN (COD) | *Le facteur distribue le courrier.* |
| GV = V + GNP (COI) | *Le facteur parle à sa collègue.* |
| GV = V + GN (COD) + GNP (COI) | *Le facteur donne une lettre à ma voisine.* |
| GV = V + GNP (COI) + GNP (COI) | *Le facteur parle de ses vacances à sa collègue.* |

#### 2.2.1 Complément d'objet direct (COD)

- Lié au sens du verbe (impliqué par le verbe)
- Fait partie du GV (ne peut pas être déplacé en tête de phrase)
- Peut être remplacé par un pronom : *Il le distribue.*
- Nature possible : GN, pronom, infinitif, proposition subordonnée complétive

#### 2.2.2 Complément d'objet indirect (COI)

- GN précédé d'une préposition (groupe nominal prépositionnel = GNP)
- Lié au sens du verbe
- Peut être remplacé par un pronom : *Le facteur lui parle.*
- Nature possible : GNP, pronom, infinitif précédé d'une préposition, proposition subordonnée

> **Note Belgique / Québec** : Le COD y est nommé « CDV » (complément direct du verbe) et le COI « CIV » (complément indirect du verbe). Ces termes ne sont **pas** ceux de la terminologie officielle française.

#### 2.2.3 Attribut du sujet

- Fonction associée aux **verbes attributifs** (être, sembler, paraître, devenir, rester, demeurer, avoir l'air, passer pour...)
- Le principal verbe attributif est **être**
- Nature possible : adjectif, GN, GNP, pronom, infinitif, proposition subordonnée conjonctive

| Exemple | Attribut |
|---|---|
| *Alice est grande.* | grande (adjectif) |
| *Alice est une bonne avocate.* | une bonne avocate (GN) |
| *Alice est en retard.* | en retard (GNP) |

#### 2.2.4 Attribut du COD

- GV comprenant au moins trois constituants : V + GN (COD) + attribut du COD
- Exemples : *On a élu Alice présidente.* / *Elles estiment Alice compétente.*

### 2.3 Le complément circonstanciel

#### Caractéristiques

- N'entretient **aucun rapport de sens nécessaire** avec le verbe
- Donne des informations complémentaires sur l'événement décrit par [GS + GV]
- Toujours **facultatif** (supprimable)
- **Déplaçable** dans la phrase

#### Types de compléments circonstanciels

| Type | Exemple |
|---|---|
| **Temps** | *Le facteur distribue le courrier **le matin**.* |
| **Lieu** | *Le facteur distribue le courrier **dans le quartier**.* |
| **Cause** | *Le courrier n'a pas été distribué **à cause des conditions météorologiques**.* |
| **Conséquence** | (niveau II) |
| **But** | (niveau II) |
| **Moyen** | (niveau II) |
| **Manière** | (niveau II) |
| **Accompagnement** | (niveau II) |
| **Concession** | (niveau II) |
| **Hypothèse** | (niveau II) |

#### Nature du complément circonstanciel

| Nature | Exemple |
|---|---|
| GNP | *à huit heures* |
| GN | *le matin* |
| Adverbe | *aujourd'hui* |
| Proposition subordonnée circonstancielle | *quand le jour se lève* |
| Gérondif | *en se dépêchant* |
| Groupe infinitif prépositionnel | *avant de partir* |

### 2.4 Les fonctions dans le groupe nominal

#### Expansion du nom

Le terme **« expansion »** est un terme général qui regroupe les fonctions épithète et complément du nom.

**Formule du GN** : GN = Dét + N (+ Exp)

#### 2.4.1 Fonction épithète

- Relation entre un **adjectif** et le nom auquel il se rapporte
- L'adjectif peut être antéposé ou postposé
- Exemple : *une **bonne** avocate* / *une avocate **remarquable***
- Peut aussi être assurée par une proposition subordonnée relative adjective (épithète) : *Les élèves **qui avaient bien révisé** ont obtenu d'excellentes notes.*

#### 2.4.2 Fonction complément du nom

- Relation entre un GNP et le nom dont il est complément
- Exemple : *la pharmacienne **de mon quartier***
- Peut aussi être assurée par : GNP, groupe infinitif prépositionnel, groupe adverbial prépositionnel, proposition subordonnée conjonctive, proposition subordonnée relative substantive

#### 2.4.3 Fonction complément de l'adjectif

- Exemple : *Elle est heureuse **de sa réussite**.*
- Nature : GNP, adverbe

#### 2.4.4 Fonction apposition

- Élément **détaché**, séparé par une virgule (ou deux virgules)
- Nature possible : adjectif, GN, proposition subordonnée relative adjective
- Exemple : *Cette avocate, **remarquable**, a gagné le procès.*
- Exemple : *Les élèves, **qui avaient bien révisé**, ont obtenu d'excellentes notes.* (relative apposée, avec virgules = tous les élèves)

> **Distinction capitale** relative épithète vs relative apposée :
> - *Les élèves qui avaient bien révisé* (sans virgules) = seulement ceux qui avaient révisé (restrictive / déterminative)
> - *Les élèves, qui avaient bien révisé,* (avec virgules) = tous les élèves (non restrictive / apposée)

### 2.5 Le complément d'agent

- Fonction propre à la **forme passive**
- Introduit par la préposition **par** (parfois **de**)
- Exemple : *La scène a été racontée **par deux témoins**.*

### 2.6 Fonctions énonciatives et textuelles (niveau II)

| Fonction | Nature | Exemple |
|---|---|---|
| Balise textuelle | Adverbe | *Premièrement, ... deuxièmement, ... enfin...* |
| Modalisateur | Adverbe | *Bizarrement, elle a agi.* |
| Connecteur | Adverbe | *Pourtant, je n'ai pas envie de sortir.* |
| Auto-énonciatif | Adverbe | *Franchement, je suis scandalisé.* |

## 3. Phrase et proposition

### 3.1 Définitions

- **Phrase** (définition graphique) : unité de sens commençant par une majuscule et se terminant par un point (ou ?, !, ...).
- **Proposition** : structure formée d'un GS et d'un GV (et éventuellement d'un GC). Une phrase peut contenir une ou plusieurs propositions.

### 3.2 Phrase simple et phrase complexe

| Type | Définition | Exemple |
|---|---|---|
| **Phrase simple** | Une seule proposition | *Le facteur distribue le courrier à huit heures.* |
| **Phrase complexe** | Au moins deux propositions | *Le facteur distribue le courrier quand le jour se lève.* |

### 3.3 Les trois modes de construction de la phrase complexe

#### 3.3.1 Subordination

- Une proposition subordonnée est **incluse** dans une proposition principale
- Introduite par un **mot subordonnant** (conjonction de subordination, pronom relatif, mot interrogatif)
- Exemple : *Le facteur distribue le courrier **quand le jour se lève**.* (prop. principale + prop. subordonnée)

#### 3.3.2 Coordination

- Deux propositions **indépendantes** reliées par une **conjonction de coordination**
- Les propositions sont sur le **même plan**
- Exemple : *Le facteur distribue le courrier **et** il aime son travail.*

#### 3.3.3 Juxtaposition

- Deux propositions indépendantes reliées par une **virgule**, un **point-virgule** ou des **deux-points**
- Exemple : *Le facteur distribue le courrier ; il aime son travail.*

### 3.4 Les propositions subordonnées (typologie détaillée, niveau II)

| Type | Sous-type | Exemple |
|---|---|---|
| **Subordonnée complétive** | Conjonctive | *Je sais **qu'il viendra**.* |
| | Interrogative totale | *Je me demande **s'il viendra**.* |
| | Interrogative partielle | *Je me demande **pourquoi il est venu**.* |
| **Subordonnée circonstancielle** | Temps, cause, conséquence, but, concession, hypothèse, comparaison | *Il sort **quand le soleil brille**.* |
| **Subordonnée relative** | Adjective (épithète) | *Le livre **qui est sur la table** est à moi.* |
| | Adjective (apposée) | *Ce livre, **qui est passionnant**, se vend bien.* |
| | Substantive | ***Qui se ressemble** s'assemble.* |
| **Subordonnée infinitive** | — | *J'entends **les enfants jouer**.* |
| **Subordonnée participiale** | — | ***Le travail achevé**, le facteur rentre chez lui.* |

### 3.5 Proposition indépendante, principale, subordonnée

| Terme | Définition |
|---|---|
| **Proposition indépendante** | Proposition qui ne dépend d'aucune autre et dont aucune autre ne dépend ; reliée à une autre par coordination ou juxtaposition |
| **Proposition principale** | Proposition qui contient (inclut) une proposition subordonnée |
| **Proposition subordonnée** | Proposition incluse dans une proposition principale |

## 4. Types et formes de phrases

### 4.1 Les trois types de phrases (obligatoires)

Toute phrase appartient **nécessairement** à l'un de ces trois types :

| Type | Acte de langage | Exemples |
|---|---|---|
| **Déclaratif** | Assertion (affirmer quelque chose, le présenter comme vrai) | *Cette chimiste travaille dans un laboratoire.* |
| **Interrogatif** | Interrogation (poser une question) | *Avez-vous terminé ? Est-ce que vous avez terminé ?* |
| **Impératif** | Injonction (donner un ordre) | *Ouvrez la fenêtre. Qu'il sorte.* |

> **Point crucial** : L'**exclamation n'est pas un type** de phrase mais une **forme** de phrase (voir ci-dessous). C'est une nuance susceptible de s'ajouter à l'un des trois types.

#### Interrogation : distinctions importantes

| Distinction | Catégories | Exemples |
|---|---|---|
| Directe / Indirecte | Interrogation directe vs indirecte | *Viendras-tu ?* vs *Je me demande si tu viendras.* |
| Totale / Partielle | Interrogation totale (oui/non) vs partielle (mot interrogatif) | *Avez-vous terminé ?* vs *Qui a parlé ?* |

### 4.2 Les formes de phrases (facultatives, cumulables)

La phrase de base est : positive, active, non exclamative. Les formes de phrase sont des transformations appliquées à cette base.

| Forme | Exemple |
|---|---|
| **Forme négative** | *Elles ne mangent pas de fraises.* |
| **Forme passive** | *La scène a été racontée par deux témoins.* |
| **Forme exclamative** | *C'est bon ! Que c'est bon !* |
| **Forme emphatique** | *C'est Paul **qui** a gagné.* (clivage) / *Paul, **lui**, a gagné.* (détachement) |
| **Forme impersonnelle** | *Il pleut. Il faut partir.* |

> **On ne dit pas** « forme positive » ni « forme active » : la phrase positive, active et non exclamative est la **phrase de base**.

### 4.3 Les phrases atypiques (non verbales)

| Type | Définition | Exemple |
|---|---|---|
| **Phrase averbale** | Phrase sans verbe conjugué | *Excellent, ce rôti !* |
| **Phrase elliptique** | Phrase dont un élément est sous-entendu | *Moi aussi.* |
| **Mot-phrase** | Un seul mot constituant une phrase | *Oui. Non. Hélas !* |

## 5. Les classes de mots (natures)

### Principe fondamental

Il existe **huit classes (natures) de mots**, réparties en deux catégories :

| Mots lexicaux | Mots grammaticaux |
|---|---|
| Nom | Déterminant |
| Verbe | Pronom |
| Adjectif | Conjonction |
| Adverbe | Préposition |

> L'**interjection** est parfois ajoutée à cette liste comme neuvième classe.
>
> Les termes **« nature »** et **« classe (grammaticale) »** sont synonymes dans la terminologie officielle.

### 5.1 Le nom

#### Catégories fondamentales

| Catégorie | Exemples |
|---|---|
| **Nom propre** | Strasbourg, la Tour Eiffel, Cendrillon |
| **Nom commun** | homme, femme, voiture, échelle |

#### Sous-catégories des noms communs

| Sous-catégorie | Exemples |
|---|---|
| Êtres animés | homme, animal, cheval |
| Êtres inanimés | table, chaise, voiture |
| Noms concrets | maison, livre |
| Noms abstraits | courage, liberté |
| Noms comptables | un livre, deux livres |
| Noms massifs | du pain, du courage |

Le nom possède un **genre** (masculin ou féminin) et varie en **nombre** (singulier ou pluriel).

### 5.2 Le déterminant

Le déterminant est un **mot grammatical** placé **avant le nom** dans le GN. Il contribue à l'identification du nom.

#### Les types de déterminants

| Type | Formes | Exemples |
|---|---|---|
| **Article défini** | le, la, l', les | *le temps, la vie, l'école, les hommes* |
| **Article indéfini** | un, une, des (de) | *un homme, une femme, des fruits* |
| **Article partitif** | du, de la, de l' | *du pain, de la farine, de l'eau* |
| **Article défini contracté** | au (= à + le), aux (= à + les), du (= de + le), des (= de + les) | *au travail, aux animaux, du travail, des champs* |
| **Déterminant possessif** | mon/ma/mes, ton/ta/tes, son/sa/ses, notre/nos, votre/vos, leur/leurs | *mon chapeau, ses livres, leur maison* |
| **Déterminant démonstratif** | ce, cet, cette, ces | *ce garçon, cet élève, cette fille, ces arbres* |
| **Déterminant interrogatif** | quel, quelle, quels, quelles | *Quel livre ? Quelle heure est-il ?* |
| **Déterminant exclamatif** | quel, quelle, quels, quelles ; que de, combien de | *Quel désastre ! Que de monde !* |
| **Déterminant indéfini** | chaque, quelque(s), certain(s), plusieurs, tout/toute/tous/toutes, aucun(e), nul(le) | *chaque élève, plusieurs livres, tout le monde* |
| **Déterminant numéral** | deux, trois, cent, mille... | *deux heures, cent coureurs* |
| **Déterminant relatif** | lequel, laquelle, lesquels, lesquelles, duquel, auquel... | *(usage littéraire)* |

> **Confusions fréquentes à éviter** :
> - **du** article partitif (*du pain*) vs **du** article défini contracté (*du travail = de le travail*)
> - **des** article indéfini pluriel (*des fruits*) vs **des** article défini contracté (*des champs = de les champs*)
> - **leur** déterminant possessif (*leur maison*) vs **leur** pronom personnel (*elle leur parle*)
> - **quel** déterminant interrogatif (*Quel livre ?*) vs **quel** déterminant exclamatif (*Quel désastre !*)

### 5.3 L'adjectif

- Exprime une **qualité** ou une **propriété** du nom auquel il se rapporte
- S'accorde en **genre** et en **nombre** avec le nom
- Deux fonctions principales : **épithète** et **attribut**

| Fonction | Exemple |
|---|---|
| Épithète | *un **beau** cahier, une voiture **rapide*** |
| Attribut du sujet | *ce cahier est **beau*** |
| Attribut du COD | *Elles estiment Alice **compétente**.* |

#### Sous-types d'adjectifs (niveau II)

| Sous-type | Caractéristique | Exemple |
|---|---|---|
| **Adjectif qualificatif** | Exprime une qualité | *grand, petit, beau* |
| **Adjectif relationnel** | Dérive d'un nom, exprime une relation | *présidentiel, national, scolaire* |
| **Adjectif verbal** | Dérive d'un verbe (participe présent devenu adjectif) | *charmant, fatigant* |
| **Adjectif numéral ordinal** | Rang dans une série | *premier, deuxième, septième* |

#### Degrés de l'adjectif

| Degré | Exemple |
|---|---|
| Comparatif d'égalité | *aussi grand que* |
| Comparatif de supériorité | *plus grand que* |
| Comparatif d'infériorité | *moins grand que* |
| Superlatif de supériorité | *le plus grand* |
| Superlatif d'infériorité | *le moins grand* |

### 5.4 L'adverbe

- Mot **invariable**
- Peut se rapporter à un **verbe**, un **adjectif** ou un autre **adverbe**

| Type d'adverbe | Exemples |
|---|---|
| **Adverbes de temps** | maintenant, hier, bientôt, aujourd'hui |
| **Adverbes de lieu** | ici, là, ailleurs, partout |
| **Adverbes de manière** | gentiment, rapidement, fermement |
| **Adverbes interrogatifs** | pourquoi ? quand ? où ? comment ? |
| **Adverbes d'intensité** | très, si, trop, assez, peu |
| **Adverbes de négation** | ne...pas, ne...jamais, ne...plus, ne...que |
| **Adverbes textuels** | premièrement, ensuite, enfin |
| **Adverbes modalisateurs** | certainement, probablement, bizarrement |
| **Adverbes énonciatifs / auto-énonciatifs** | franchement, sincèrement |

> **Suffixe -ment** : Les adverbes de manière en *-ment* sont formés sur le féminin de l'adjectif + *-ment* (issu du latin *mente* = « manière d'être »). Ex. : *finalement, anciennement*.

### 5.5 La préposition

- Mot **invariable**
- Introduit un **groupe prépositionnel** qui joue un rôle de complément
- Se place devant un GN, un pronom, un adverbe ou un infinitif

**Les prépositions les plus fréquentes** (par ordre de fréquence) :
**de, à, en, dans, pour, sur, avec, par, sans**

Les prépositions peuvent être constituées d'un mot (*sur*) ou de plusieurs (*au-dessus de*) : on parle alors de **locution prépositionnelle**.

#### Types de groupes prépositionnels

| Type | Abréviation | Exemple |
|---|---|---|
| Groupe nominal prépositionnel | GNP | *de mon quartier* |
| Groupe infinitif prépositionnel | GIP | *pour travailler* |
| Groupe adverbial prépositionnel | GAP | *d'aujourd'hui* |
| Groupe pronominal prépositionnel | GPP | *pour toi* |

### 5.6 Le pronom

Le pronom « tient lieu d'un GN » (il est préférable de dire cela plutôt que de dire qu'il « remplace un nom »).

#### 5.6.1 Le pronom personnel

| Fonction | Formes |
|---|---|
| **Sujet** | je, tu, il, elle, nous, vous, ils, elles, on |
| **COD** | me, te, le, la, nous, vous, les, se |
| **COI** | me, te, lui, nous, vous, leur, se |
| **Formes disjointes** (après préposition) | moi, toi, soi, lui, elle, nous, vous, eux, elles |

> **on** : pronom indéfini pouvant être employé comme pronom personnel sujet (équivalent de *nous*).

#### 5.6.2 Les autres types de pronoms

| Type | Exemples |
|---|---|
| **Pronom démonstratif** | celui, celle, ceux, celles, celui-ci, celui-là, ce, ceci, cela, ça |
| **Pronom possessif** | le mien, la mienne, les miens, le tien, le sien, le nôtre, le vôtre, le leur... |
| **Pronom indéfini** | on, tout, chacun, personne, rien, quelqu'un, certains, plusieurs, aucun... |
| **Pronom interrogatif** | qui ? que ? quoi ? lequel ? laquelle ?... |
| **Pronom relatif** | qui, que, quoi, dont, où, lequel, laquelle, auquel, duquel... |
| **Pronom adverbial** | en, y |

> **Pronoms adverbiaux** *en* et *y* : Ils fonctionnent comme des pronoms mais ont une forme adverbiale.
> - *en* = COI : *Elle en parle.* (= de ses vacances)
> - *y* = complément circonstanciel ou COI : *Elle y va.* (= à Paris)

### 5.7 La conjonction

Mot **invariable**. Deux types :

#### 5.7.1 Conjonction de coordination

Relie des mots, des groupes de mots ou des propositions **de même fonction** (et souvent de même nature).

**Liste officielle** : **mais, ou, et, or, ni, car**

> **Attention** : Le mot **donc** est considéré comme un **adverbe** (et non une conjonction de coordination) dans la terminologie officielle. De même pour *pourtant, cependant*, etc.

#### 5.7.2 Conjonction de subordination

Mot invariable qui permet à une proposition subordonnée d'être incluse dans la proposition principale.

| Sens | Exemples |
|---|---|
| Temps | quand, lorsque, dès que, avant que, après que, pendant que, tandis que |
| Cause | parce que, puisque, comme, sous prétexte que |
| Conséquence | si bien que, de sorte que, au point que |
| But | pour que, afin que, de peur que |
| Concession | bien que, quoique, même si |
| Hypothèse | si, à condition que, pourvu que |
| Comparaison | comme, de même que, ainsi que |

> **que** est la conjonction de subordination fondamentale ; les locutions conjonctives (parce que, dès que, etc.) sont formées sur *que*.

### 5.8 L'interjection et l'onomatopée

| Classe | Exemples |
|---|---|
| **Interjection** | Ah ! Oh ! Hélas ! Bravo ! Zut ! |
| **Onomatopée** | Boum ! Tic-tac ! Miaou ! Cocorico ! |

## 6. Le verbe et la conjugaison

### 6.1 Morphologie verbale : radical et désinence

| Élément | Définition | Exemple avec *chantaient* |
|---|---|---|
| **Radical** | Porteur du sens lexical stable | chant- |
| **Désinence** (ou terminaison) | Varie selon le temps, le mode et la personne | -aient |
| **Marque de temps** | Partie de la désinence indiquant le temps | -ai- (marque de l'imparfait) |
| **Marque de personne** | Partie de la désinence indiquant la personne | -ent (marque de la 3e personne du pluriel) |

#### Nombre de radicaux

| Verbe | Radicaux | Exemples |
|---|---|---|
| chanter | 1 radical | chant- |
| lire | 2 radicaux | li- (je lis), lis- (nous lisons) |
| vivre | 3 radicaux | vi- (je vis), viv- (je vivais), véc- (je vécus) |
| être, avoir, aller, pouvoir, savoir | plus de 3 radicaux | — |

### 6.2 Les trois groupes de verbes

| Groupe | Critère | Exemples |
|---|---|---|
| **1er groupe** | Infinitif en **-er** ; un seul radical | chanter, parler, aimer |
| **2e groupe** | Infinitif en **-ir** ; participe présent en **-issant** | finir (finissant), bondir, grossir |
| **3e groupe** | Tous les autres verbes (catégorie hétérogène) | partir, savoir, écrire, aller, être, avoir |

#### Sous-types du 3e groupe

| Sous-type | Exemples |
|---|---|
| Infinitif en -ir (sans pp en -issant) | partir, servir, fuir |
| Infinitif en -oir | savoir, apercevoir, devoir |
| Infinitif en -re | écrire, dire, prendre |
| Verbe *aller* | aller |
| Verbes *être* et *avoir* | être, avoir |

> **Attention** : Le verbe *aller* n'est **pas** du 1er groupe malgré son infinitif en -er (il a plusieurs radicaux).

### 6.3 Les modes

#### 6.3.1 Modes personnels (varient selon la personne)

| Mode | Caractéristique |
|---|---|
| **Indicatif** | Mode de l'assertion, le plus utilisé |
| **Subjonctif** | Mode de la subordination, du souhait, du doute |
| **Impératif** | Mode de l'injonction (ordre, conseil, prière) |

#### 6.3.2 Modes non personnels (ne varient pas selon la personne)

| Mode | Formes |
|---|---|
| **Infinitif** | aimer, finir, prendre |
| **Participe** | aimant (présent), aimé (passé) |
| **Gérondif** | en aimant, en finissant |

> **Point crucial** : L'infinitif et le participe **sont des formes de la conjugaison** du verbe. Il est **erroné** de les considérer comme « des formes non conjuguées ».

### 6.4 Les temps de l'indicatif

| | Temps simples | Temps composés |
|---|---|---|
| **Présent** | je chante | **Passé composé** : j'ai chanté |
| **Imparfait** | je chantais | **Plus-que-parfait** : j'avais chanté |
| **Passé simple** | je chantai | **Passé antérieur** : j'eus chanté |
| **Futur** | je chanterai | **Futur antérieur** : j'aurai chanté |
| **Conditionnel présent** | je chanterais | **Conditionnel passé** : j'aurais chanté |

> **Point crucial sur le conditionnel** : Le conditionnel doit être considéré comme un **temps de l'indicatif** (et non un mode à part entière), selon la terminologie officielle. Morphologiquement, pour les verbes du 1er groupe, le conditionnel est au futur ce que l'imparfait est au passé simple : chanter**ai** / chanter**ais** / chant**ai** / chant**ais**. Toutefois, dans l'usage scolaire, le conditionnel est **parfois** encore enseigné comme un mode.

### 6.5 Les temps du subjonctif

| Temps simples | Temps composés |
|---|---|
| **Présent** : qu'elle fasse | **Passé** : qu'elle ait fait |
| **Imparfait** : qu'elle fît | **Plus-que-parfait** : qu'elle eût fait |

### 6.6 Les temps de l'impératif

| Temps | Exemple |
|---|---|
| **Présent** | Chante ! Chantons ! Chantez ! |
| **Passé** | Aie fini ! Ayons fini ! Ayez fini ! |

> **Rappel orthographique** : Pas de -s à la 2e personne du singulier des verbes du 1er groupe à l'impératif : *Chante ! Traverse !* (sauf devant *en* et *y* : *Manges-en !*).

### 6.7 Auxiliaires et semi-auxiliaires

| Catégorie | Verbes | Rôle |
|---|---|---|
| **Auxiliaires** | être, avoir | Formation des temps composés et de la voix passive |
| **Semi-auxiliaires** | aller, venir de, devoir, pouvoir, faire (+ infinitif) | Valeurs aspectuelles, modales ou causatives |

### 6.8 Les voix

| Voix | Définition | Exemple |
|---|---|---|
| **Voix active** | Le sujet fait l'action | *Le chat attrape la souris.* |
| **Voix passive** | Le sujet subit l'action | *La souris est attrapée par le chat.* |
| **Voix pronominale** | Le verbe est accompagné d'un pronom réfléchi | *Elle se regarde.* |
| **Voix impersonnelle** | Sujet apparent « il » sans référent | *Il pleut. Il faut partir.* |
| **Voix factitive** | Le sujet fait faire l'action | *Elle fait construire une maison.* |

#### Sous-types de verbes pronominaux

| Sous-type | Exemple |
|---|---|
| Réfléchi (réflexif) | *Elle se regarde.* (= elle-même) |
| Réciproque | *Ils se parlent.* (= l'un à l'autre) |
| À sens passif | *Ce livre se vend bien.* |
| À sens actif (essentiellement pronominal) | *Elle s'évanouit.* |

### 6.9 Transitivité du verbe

| Type | Définition | Exemple |
|---|---|---|
| **Transitif direct** | Admet un COD | *Elle lit un livre.* |
| **Transitif indirect** | Admet un COI | *Elle parle de ses vacances.* |
| **Intransitif** | N'admet ni COD ni COI | *L'enfant dort.* |
| **Attributif** | Suivi d'un attribut | *Elle est avocate.* |

## 7. Le lexique

### 7.1 Mots simples et mots complexes

| Type | Définition | Exemples |
|---|---|---|
| **Mot simple** | Un seul morphème lexical, sans préfixe ni suffixe | souris, table, maison, faire |
| **Mot complexe** | Plusieurs morphèmes | refaire, national, portefeuille |

### 7.2 La formation des mots (morphologie lexicale)

#### 7.2.1 Dérivation

| Procédé | Définition | Exemple |
|---|---|---|
| **Préfixation** | Ajout d'un préfixe avant le radical ; ne change **pas** la classe grammaticale | capable → incapable (adj. → adj.) |
| **Suffixation** | Ajout d'un suffixe après le radical ; **peut changer** la classe grammaticale | président (N) → présidentiel (Adj.) |
| **Parasynthétique** | Préfixation et suffixation simultanées (ni le préfixé seul ni le suffixé seul n'existe) | encolure, dévitaliser |

#### Précisions : base, radical, racine

| Terme | Définition | Exemple |
|---|---|---|
| **Base** | Mot servant de point de départ à la dérivation | *nation* est la base de *national* |
| **Radical** | Forme que prend la base en construction avec un affixe | *chant-* est le radical de *chanteur* |
| **Racine** | Élément étymologique (souvent latin ou grec) | *spir-* (du latin *spirare*) dans respirer, inspirer, expirer |

#### 7.2.2 Composition

Les **mots composés** sont formés de deux unités lexicales autonomes.

| Formation | Exemples |
|---|---|
| nom + nom | poisson-chat, porte-fenêtre |
| nom + préposition + nom | pomme de terre, arc-en-ciel |
| nom + adjectif | coffre-fort, château fort |
| adjectif + nom | rouge-gorge |
| verbe + nom | portefeuille, porte-clés |
| adjectif + adjectif | aigre-doux |
| adverbe + adjectif | tout-puissant |
| préposition + nom | avant-projet |
| préposition + verbe | sous-entendre |

#### 7.2.3 Autres procédés de formation

| Procédé | Définition | Exemple |
|---|---|---|
| **Conversion** | Changement de classe sans modification de forme | *le dîner* (verbe → nom) |
| **Sigles et acronymes** | Formation par initiales | SNCF, ONU, sida |
| **Troncation** | Abrègement d'un mot | métro (métropolitain), prof (professeur) |
| **Mot-valise** | Fusion de deux mots | courriel (courrier + électronique) |
| **Emprunt** | Mot issu d'une autre langue | parking, week-end |

### 7.3 Relations sémantiques entre les mots

| Relation | Définition | Exemples |
|---|---|---|
| **Famille de mots** | Ensemble de mots partageant un même radical | cœur, cardiaque, cordial, cordialité |
| **Champ lexical** | Ensemble de mots se rapportant à un même thème dans un texte | nature : prairie, herbe, forêt, branche, feuille... |
| **Synonymie** | Mots de sens proches, interchangeables dans certains contextes | jouer / s'amuser ; logement / habitation |
| **Antonymie** | Mots de sens opposés | vivant / mort ; chaud / froid ; grand / petit |
| **Hyperonymie** | Mot de sens plus général englobant un autre | *animal* est hyperonyme de *chat* |
| **Hyponymie** | Mot de sens plus spécifique inclus dans un autre | *chat* est hyponyme d'*animal* |
| **Homonymie** | Mots de significations différentes mais de même forme | mousse (plante) / mousse (marin) |
| **Polysémie** | Un même mot possédant plusieurs sens liés | souris (rongeur / informatique) |

#### Homonymes : sous-catégories

| Type | Définition | Exemple |
|---|---|---|
| **Homonymes homographes** | Même écriture, sens différents | les fils (de mon frère) / les fils (de laine) |
| **Homonymes homophones** | Même prononciation, écriture et sens différents | vers / vert / verre |

> **Distinction polysémie / homonymie** : La polysémie concerne les **différents sens d'un même mot** (liés par une similarité), tandis que l'homonymie concerne des **mots différents** qui se trouvent avoir la même forme.

#### Registres de langue

| Registre | Exemple |
|---|---|
| **Familier** | canasson, bagnole, bouquin |
| **Courant** | cheval, voiture, livre |
| **Soutenu** | destrier, véhicule, ouvrage |

> Le terme **« registre de langue »** peut aussi être nommé **« niveau de langue »** dans l'usage scolaire.

### 7.4 Dénotation et connotation (niveau II)

| Concept | Définition |
|---|---|
| **Dénotation** | Sens objectif, référentiel d'un mot |
| **Connotation** | Valeurs subjectives, affectives ou culturelles associées à un mot |

### 7.5 Évolution sémantique des mots (niveau II)

| Procédé | Définition | Exemple |
|---|---|---|
| **Sens premier** | Sens originel du mot | souris = rongeur |
| **Sens par analogie / métaphore** | Transfert de sens par ressemblance | souris d'ordinateur |
| **Sens par métonymie** | Transfert de sens par contiguïté | boire un verre |
| **Extension** | Élargissement du sens | panier (objet → panier de basket) |
| **Restriction** | Rétrécissement du sens | viande (toute nourriture en ancien français → chair animale) |

## 8. Points de vigilance terminologique pour le CRPE

### 8.1 Termes officiels vs termes non recommandés

| Terme officiel (recommandé) | Terme déconseillé ou à nuancer | Explication |
|---|---|---|
| **Complément d'objet direct (COD)** | CDV (complément direct du verbe) | CDV est utilisé en Belgique et au Québec, pas en France |
| **Complément d'objet indirect (COI)** | CIV (complément indirect du verbe) | CIV est utilisé en Belgique et au Québec, pas en France |
| **Désinence** (ou terminaison) | — | Les deux termes sont acceptés ; « terminaison » est l'équivalent en classe |
| **Nature** = **classe grammaticale** | — | Les deux termes sont officiellement synonymes |
| **Conditionnel = temps de l'indicatif** | Conditionnel = mode | La terminologie officielle le classe comme temps de l'indicatif, mais reconnaît que l'usage scolaire le traite parfois comme mode |
| **Exclamation = forme de phrase** | Exclamation = type de phrase | L'exclamation n'est pas un 4e type ; c'est une forme |
| **Phrase de base** (positive, active, non exclamative) | Forme positive, forme active | On ne parle pas de « forme positive » ou « forme active » |
| **donc** = adverbe | donc = conjonction de coordination | La terminologie officielle classe *donc* comme adverbe (il peut apparaître en fin de phrase : *Tu pars, donc.*) |
| **Le pronom tient lieu d'un GN** | Le pronom remplace un nom | Formulation plus précise |
| **Expansion du nom** | — | Terme général englobant épithète et complément du nom |
| **Groupe sujet (GS)** | — | Peut être réduit à un seul mot (pronom, nom propre) |
| **Prédicat** | — | Terme d'intérêt limité car le GV a toujours la même fonction (il est le pivot de la phrase) ; son usage est mentionné mais non imposé |
| **Infinitif et participe = formes conjuguées** | Formes non conjuguées | Ce sont des modes de la conjugaison (radical + désinence) |
| **Verbe attributif** | Verbe d'état | Le terme officiel est « verbe attributif » |

### 8.2 Principes d'analyse à retenir

1. **La phrase est la structure fondamentale** ; les autres structures sont définies par rapport à elle.
2. **Comprendre la grammaire = comprendre les relations d'inclusion** (une subordonnée est incluse dans une principale, un complément du nom est inclus dans un GN, etc.).
3. **Nature vs Fonction** : l'essentiel de l'analyse grammaticale scolaire consiste à identifier la **nature** (ce qu'est un mot/groupe de mots) et la **fonction** (son rôle dans la phrase).
4. **Les mots grammaticaux (déterminants, prépositions, conjonctions) n'ont pas de fonction par eux-mêmes** ; ils font partie d'un groupe qui a une fonction.
5. **Le verbe/GV est le pivot de la phrase** : il n'a pas de terme spécifique pour sa fonction.
6. **L'ordre des mots** est essentiel en français (à la différence du latin et de son système de déclinaisons).

### 8.3 Les deux niveaux de la terminologie

| Niveau I | Niveau II |
|---|---|
| Description d'ensemble minimale | Description plus complète et approfondie |
| Structures canoniques les plus simples | Cas particuliers, nuances, développements théoriques |
| Suffisant pour enseigner en école primaire | Connaissances approfondies pour la formation des enseignants |

## 10. Tableaux de synthèse

### 10.1 Les natures possibles pour chaque fonction

| Fonction | Natures possibles |
|---|---|
| **Sujet** | GN, pronom (personnel, démonstratif, possessif, indéfini, interrogatif, relatif), infinitif, groupe infinitif, prop. sub. conjonctive, prop. sub. relative substantive, prop. sub. interrogative partielle |
| **COD** | GN, pronom, infinitif, groupe infinitif, prop. sub. conjonctive, prop. sub. relative substantive, prop. sub. interrogative (totale et partielle), prop. sub. infinitive |
| **COI** | GNP, pronom, pronom adverbial, infinitif (+ prép.), groupe infinitif (+ prép.), prop. sub. conjonctive (+ prép.), prop. sub. relative substantive, prop. sub. interrogative (totale et partielle) |
| **Attribut du sujet** | Adjectif, GAdj, GN, GNP, pronom, infinitif, prop. sub. conjonctive, prop. sub. relative substantive, prop. sub. interrogative partielle |
| **Attribut du COD** | Adjectif, GAdj, GN, GNP, prop. sub. relative adjective |
| **Complément circonstanciel** | Adverbe, GNP, GN, gérondif, groupe infinitif prépositionnel, groupe pronominal prépositionnel, prop. sub. circonstancielle, prop. sub. participiale |
| **Complément du nom** | GNP, groupe pronominal prépositionnel, groupe infinitif prépositionnel, groupe adverbial prépositionnel, prop. sub. conjonctive, prop. sub. relative substantive |
| **Épithète** | Adjectif, GAdj, GN (réduit au nom commun ou propre), prop. sub. relative adjective |
| **Complément de l'adjectif** | GNP, adverbe |
| **Apposition** | Adjectif, GAdj, GN, prop. sub. relative adjective |
| **Complément d'agent** | GNP, pronom relatif |

### 10.2 Les huit classes de mots — résumé

| Classe | Variable / Invariable | Mot lexical / grammatical | Fonctions possibles |
|---|---|---|---|
| **Nom** | Variable (genre, nombre) | Lexical | Sujet, COD, attribut, épithète (nom en apposition), complément du nom, apposition |
| **Verbe** | Variable (mode, temps, personne, nombre) | Lexical | Pivot de la phrase (pas de terme de fonction propre) |
| **Adjectif** | Variable (genre, nombre) | Lexical | Épithète, attribut du sujet, attribut du COD, apposition |
| **Adverbe** | Invariable | Lexical | CC, complément de l'adjectif, complément du verbe, connecteur, modalisateur |
| **Déterminant** | Variable (genre, nombre selon les types) | Grammatical | Pas de fonction propre (fait partie du GN) |
| **Pronom** | Variable (genre, nombre, personne, fonction) | Grammatical | Sujet, COD, COI, attribut |
| **Conjonction** | Invariable | Grammatical | Pas de fonction propre (outil de liaison) |
| **Préposition** | Invariable | Grammatical | Pas de fonction propre (introduit un GP) |
